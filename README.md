# ReactScheduler

## Description

React.jsを使用したスケジューラーアプリ  

## Installation

    $ git clone https://github.com/ArrowDevLab/ReactScheduler.git  

## フォルダー構成

    ReactScheduler  
    + docker
        + db
        + node
            + app
            + Dockerfile
        + docker-compose.yml
    + provision_init.sh
    + provision_base.sh
    + Vagrantfile
    + README.md

## Building

    $ cd ReactScheduler  
    $ vagrant up

システム起動後にブラウザで`http://localhost:3001`と入力する
